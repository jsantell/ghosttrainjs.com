GhostTrain

Mock Router

GhostTrain is a mock router library for serving mock data statically. Leveraging the same API as Express, GhostTrain allows developers to send mock requests to GhostTrain rather than making HTTP requests to a server, for development, testing and demos. Run in a browser, in node, and take development to the next level.

  var gt = new GhostTrain();
  var = users { '12': myUser };
  gt.get('/users/:id', function (req, res) {
    res.send(users[req.params.id]);
  });

  gt.request('get', '/users/12', function (e, res, body) {
    console.log(res.statusCode); // 200
    console.log(body); // User Object
  });

Speedy Development

Drop GhostTrain into your web app to mock API responses so you don't have to spin up your API server, and so you don't have to depend on server features being incomplete or broken.

App Demos

Turn your web app service into a demo by using GhostTrain's mock data to populate the user experience you want to show off without ever pinging your servers.

Testing

Run tests against GhostTrain's mock API rather than a remote server to ensure the durability of your code, separate from the service's API.

Reuse Router Code

GhostTrain's routing API is based on Express, so if you're using Express, you can reuse your router code with a changed data store for your client.
